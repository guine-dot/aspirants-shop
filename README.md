# ASPIRANTS.SHOP - Premium Game Top-Up Platform

A premium game top-up website that allows users to purchase in-game currency and packages for popular mobile games.

## Features

- **User Authentication**: Email/phone verification, secure login, password recovery
- **Game Top-Up System**: Multiple games with customizable packages
- **Theme Customization**: 9 premium themes with live switching
- **Payment Integration**: QR code and UPI payment system
- **Order Management**: Order history, status tracking, WhatsApp integration
- **Advanced Admin Panel**: Complete site management with staff permissions, analytics, and more
- **Real-time Analytics**: Dashboard with comprehensive metrics

## Tech Stack

### Frontend
- React 18 with TypeScript
- Tailwind CSS
- Redux for state management
- Axios for API calls

### Backend
- Node.js with Express
- MongoDB
- JWT Authentication
- Nodemailer for emails

## Installation

### Prerequisites
- Node.js (v16+)
- MongoDB
- npm or yarn

### Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   cd client && npm install
   ```

3. Create `.env` file (see `.env.example`)

4. Start development server:
   ```bash
   npm run dev
   ```
